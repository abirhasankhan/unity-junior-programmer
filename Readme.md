# Unity Junior Programmer 🎮💻

This repository contains my learning projects, challenges, and exercises from the **Unity Junior Programmer** pathway.

I'm using this space to practice **C# programming**, learn **game development with Unity**, and document my progress as I work toward becoming a Unity developer.

## 🚀 About the Pathway

The Junior Programmer pathway is designed for beginners who want to learn programming with Unity and build the skills needed for entry-level Unity development roles. It focuses on coding fundamentals, object-oriented programming, debugging, project management, and creating interactive applications.

* Object-Oriented Programming (OOP)
* Unity scripting and APIs

* UI programming

* Code optimization and best practices
